# Maulik Makwana - Professional Portfolio

This is the professional portfolio website for Maulik Makwana, a Senior Software Engineer.

## Features
- **Professional Design:** Clean and corporate aesthetic.
- **Theme Toggle:** Switch between Professional (Light), Dark, and Creative themes.
- **Responsive:** Works on all devices.

## Deployment
This site is ready to be hosted on GitHub Pages.

1. Create a new repository on GitHub.
2. Push this code to the repository.
3. Go to Settings > Pages and enable GitHub Pages from the `main` branch.
